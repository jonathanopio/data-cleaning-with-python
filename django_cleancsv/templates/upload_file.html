What makes data clean?

Uniqueness

Calculate the percentage of unique values in each column. This can be an indicator of the diversity and variability of the data, as well as identify the %age of possible duplicates..

Consistency

Check if data in each column adheres to a consistent format (e.g., if a column is supposed to contain percentages, check that all values are in percentage format).

Completeness

Calculate the percentage of missing values in the dataset. This helps assess how complete the dataset is.
